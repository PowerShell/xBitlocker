TOPIC
    about_xBitlocker

SHORT DESCRIPTION
    This DSC module allows you to configure Bitlocker.

LONG DESCRIPTION
    This DSC module allows you to configure Bitlocker on a single disk, configure
    a TPM chip, or automatically enable Bitlocker on multiple disks.

EXAMPLES
    PS C:\> Get-DscResource -Module xBitlocker

NOTE:
    Thank you to the DSC Community contributors who contributed to this module by
    writing code, sharing opinions, and provided feedback.
